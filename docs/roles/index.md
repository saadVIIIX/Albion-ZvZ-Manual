# Role Guides

Role guides explain what each role should do before, during, and between major fight moments.

A role page is not only a build page. A build tells you what you wear. A role guide tells you what job you are responsible for.

## Main role groups

| Role group | What they protect or create |
|---|---|
| [All Players](all-players.md) | spacing, call interpretation, survival discipline |
| [DPS Overview](dps-overview.md) | broad DPS responsibility before weapon-specific detail |
| [Ranged DPS](ranged-dps.md) | ranged pressure, timing, and conversion |
| [Melee Pressure / Execute DPS](melee-pressure-dps.md) | melee pressure, executes, and safe commit windows |
| [Support DPS](support-dps.md) | pierce, anti-heal, purge, utility, window creation |
| [Holy Healers](holy-healers.md) | direct stabilization and burst recovery |
| [Nature Healers](nature-healers.md) | sustained coverage, zones, HoT value |
| [Supports](supports.md) | defensive layers and utility support |
| [Engage Tanks](engage-tanks.md) | clumps, locks, and engage windows |
| [Stopper Tanks](stopper-tanks.md) | denial of enemy engage paths |
| [Scouts / Information](scouts-info.md) | useful information before the zerg needs it |

## How to read a role page

Ask:

- Where should I stand?
- What do I watch?
- What am I preparing before the call?
- What mistake ruins my role?
- What related fight concept should I learn next?
