# Applied Build Cards

Build cards are **applied package references**, not the center of the public manual.

The public manual teaches general ZvZ concepts. Build cards belong to specific workbook packages and should be marked draft/workbook-backed until reviewed.

## Current active package

| Package | Status |

| Great Fire Control Formation | active workbook package, draft public build cards |

## Current build-card group

- [Great Fire Control Build Cards](control-comp/index.md)

## Important rule

Specific weapons like Great Fire, Carrioncaller, Locus, or Oathkeepers should not appear to be universal public-manual defaults.

They are valid inside their applied package.
