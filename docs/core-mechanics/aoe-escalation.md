# AoE Escalation

!!! warning "Patch-sensitive"
    Exact values may change. Verify current values with Albion Wiki, patch notes, or in-game tooltips before publishing hard numbers.


## What it means

AoE escalation is the reason clumps explode. If many players stand inside the same damage area, the enemy gets more value from every AoE spell.

## Field behavior

Do not stand on friendly players, especially near walls, chokes, doors, corners, and objective boxes.
