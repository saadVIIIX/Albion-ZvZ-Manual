# Healing Sickness

!!! warning "Patch-sensitive"
    Exact values may change. Verify current values with Albion Wiki, patch notes, or in-game tooltips before publishing hard numbers.


## What it means

Multiple healers and repeated healing interactions can reduce healing value. This is one reason healer composition and positioning matter.

## Field behavior

More healers is not automatically better if the zerg loses damage, support, and role balance.
