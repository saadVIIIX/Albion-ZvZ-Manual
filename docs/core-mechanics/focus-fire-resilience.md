# Focus Fire / Resilience

!!! warning "Patch-sensitive"
    Exact values may change. Verify current values with Albion Wiki, patch notes, or in-game tooltips before publishing hard numbers.


## What it means

Focus fire mechanics prevent some target types from dying instantly to random stacked hits. This affects how zergs choose targets and why front-to-back pressure differs from full engage damage.

## Field behavior

Pressure tanks with Q/W when appropriate, but do not waste a full zerg engage into a protected low-value target unless the situation supports it.
