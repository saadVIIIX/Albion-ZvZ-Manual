# Beta Candidate Go / No-Go Checklist

Use this before sharing Beta Candidate 1.

## Go

Beta Candidate 1 is shareable with trusted reviewers if:

- preview opens from `index.html`
- curated navigation works
- Start Here works
- role guides are reachable
- workbook is clearly optional
- source credit is clear
- known limitations are visible
- no private data is exposed
- no raw markup appears
- patch-sensitive claims are marked or softened

## No-go

Do not share if:

- the master workbook is linked as public
- Great Fire Control Formation is presented as universal doctrine
- future workbook packages appear active without support
- transcript source maps dominate the public reader path
- current item values are stated as stable without verification
- private VOD or player-specific material is exposed

## Current decision

The package is a **Beta Candidate**, not a public release.

The correct audience is a small set of trusted reviewers.
