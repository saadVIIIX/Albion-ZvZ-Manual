# Known Limitations

The manual is live and usable, but it is still growing.

## Current limits

- Some pages are more polished than others.
- Build cards are still mostly text-first.
- Item icons are not implemented yet.
- The current diagrams are teaching visuals, not final art.
- Exact mechanics values should be checked when they matter for a current build or patch-sensitive claim.
- The current Formation Ledger contains one active applied package. It should not be read as the whole manual or as a universal meta claim.
- Future comp packages should only be added when they have enough reliable data.
- Some deeper weapon/tool notes can be refined later as more source material is integrated.

## What still belongs later

- deeper integration of remaining AO University transcript batches
- stronger build-card visuals and consistent item icon policy
- more diagrams and visual examples
- broader comp-package support when reliable community data exists
- a possible move to a project GitHub organization for a cleaner free URL

## What should not block use

The manual does not need every possible Albion weapon, comp, or meta branch before being useful. Its main value is teaching role function, decision-making, movement, positioning, pressure, defensives, terrain, and review habits.
