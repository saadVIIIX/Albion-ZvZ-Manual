# Current Scope

The manual is live and usable, but it is not trying to be every Albion ZvZ resource at once.

## What this site focuses on

- role behavior
- movement and positioning
- pressure and counter-pressure
- defensive timing and healer coverage
- terrain, chokes, doors, and fight shape
- practical review habits
- the Formation Ledger as an optional operating tool

## What is intentionally limited

- The Formation Ledger currently has one active public copy: **Great Fire Control Formation**.
- Build details stay inside the Formation Ledger instead of taking over the general manual.
- Weapon examples teach function, not universal best-in-slot commands.
- Exact item values and current strength claims should be checked before they become build instructions.

## Later improvements

Future versions can add stronger visuals, deeper role examples, cleaner build-card presentation, and additional Formation Ledger packages if enough reliable data exists.

The manual does not need every possible comp, weapon, or meta branch before it is useful. Its main value is teaching players how to think and act better in ZvZ.
