# Disengage Tank Deep Dive

Episode 14 focuses on disengage tanks.

The central lesson is that a disengage tank is not just a tank standing in front. A disengage tank reads the enemy, buys time, staggers incoming damage, and protects the zerg from the moment before enemy damage lands.

## True disengage weapons discussed

| Weapon | Main identity | Key value |
|---|---|---|
| Heavy Mace | purge/silence/root disengage tank | low-cooldown purge + silence pressure, Q silence pool, snare charge |
| Grovekeeper | jump-stun / ring slow disengage tank | long stun, resist gain, ring slow, clump-control overlap |
| Grailseeker | long-line root / kite-disengage hybrid | mobility, line-root, wall extension, strong against movement paths |

## Heavy Mace notes

AOU emphasizes Heavy Mace as extremely strong because it combines several rare values:

- Q silence pool
- W snare charge for mobility, interrupt, and root
- E purge plus silence
- low cooldown relative to its fight impact
- ability to strip key offensive/defensive setups

Public-manual framing:

> Heavy Mace is not only a tank weapon; it is a timing weapon. It must interrupt the enemy plan before damage lands.

Patch caution: exact cooldowns, item interactions, and meta status should be verified before stable publication.

## Grovekeeper notes

Grovekeeper brings:

- Q pierce value
- hammer ring slow
- jump mobility
- long stun
- resistance gain during engage
- strong value when placed on the enemy path

Public-manual framing:

> Grovekeeper is valuable because it combines mobility, control, and survivability in one role.

## Grailseeker notes

Grailseeker brings:

- cartwheel mobility and resistance
- W options for stun, knock-up, or knockback-style utility
- long-line root
- option to use one or both casts depending on cooldown need
- strong wall/terrain extension

Public-manual framing:

> Grailseeker should cut across enemy movement, not run parallel with it.

## Armor logic

| Armor | Use logic |
|---|---|
| Judicator Armor | self + group resistance/heal-boost value in danger area |
| Guardian Armor | useful when Heavy Mace/Grovekeeper enters enemy body and reduces damage/speed |
| Duskweaver Armor | speed-purge/slow layer against sprinting engages; coordinate so multiple users do not overwrite each other |
| Knight Armor | choke denial and anti-push utility |

Demon Armor is specifically warned against for disengage tanks because it often requires the tank to stand inside the friendly zerg when their real job is to be between enemy engage and the friendly zerg.

## Helmet / boot logic

| Tool | Use logic |
|---|---|
| Knight Helmet | entry/exit through CC, but patch-sensitive |
| Judicator Helmet | extra interrupt/stun value while inside enemy |
| Hellion Hood | smoke/invisibility/silence value; exact interaction patch-sensitive |
| Assassin Hood | criticized because channeling can pull a disengage tank away from their job |
| Guardian Boots | high-skill survivability option that requires excellent positioning |
| Boots of Valor / Graveguard Boots | CC-immunity movement options |
| Mercenary Shoes | useful, but can teach bailout habits if relied on too much |

## Playstyle rules

### Read the enemy

A good disengage tank should often feel the enemy countdown before it happens. Signs include:

- many enemy name tags moving in one direction at once
- enemy players ignoring nearby Q/W/autos because they are preparing an engage
- enemy DPS/tanks stacking on one side
- sudden movement toward a friendly clump

### Read your own zerg

If your own zerg clumps near a wall, tree, choke, or terrain pocket, the disengage tank should already know that enemy damage is likely coming there.

### Do not over-disengage

Stop the enemy only as much as the zerg is actually in danger.

If the zerg is no longer clumped or the enemy can only kill one isolated player, do not waste every cooldown.

### Cycle cooldowns

Enemy engages may come in layers:

- bomb squad
- first party
- second party
- main zerg
- delayed follow-up

A disengage tank who blows every cooldown at once may lose the next layer.

### Hold the wall

Walls, ridges, rivers, map edges, and terrain lines are fight-shaping objects. A disengage tank should help deny enemy angle control, not only stand in the center.

### Clean the backline

If enemies are already inside the backline, a disengage tank should help clean them out. Holding a choke is meaningless if the zerg is already rotting from the inside.

## Review questions

- Did the D-tank act before enemy damage landed?
- Did they use one cooldown at a time or panic everything?
- Did they preserve wall/angle control?
- Did they return to clean the backline when needed?
- Did they stop real danger or over-disengage harmless movement?
