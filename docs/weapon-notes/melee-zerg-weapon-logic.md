# Melee Zerg Weapon Logic

This page extracts the broad melee-zerg weapon logic from AO University Ep. 8.

It separates **burst melee** from **pressure/brawler melee** so the manual does not treat all melee DPS as one thing.

## Two melee identities

| Melee style | Main idea | Failure pattern |
|---|---|---|
| Burst melee | fast synchronized damage, usually on a short engage window | no kills means it must reset or it gets countered |
| Pressure / brawler melee | sustained Q/W/DoT/soft pressure, healer strain, spread pressure | less flashy; harder to sell because value is not always instant kills |

## Burst melee tools from the episode

| Tool | Teaching value | Caution |
|---|---|---|
| Bridal Fury | burst hit with jump-back safety; can function almost like a ranged burst tool | not a flood weapon in the same way Demon Fang can be |
| Demon Fang | low-cooldown burst identity; can add extra single E pressure between full engages | players often use it too conservatively or do not exploit cooldown properly |
| Raven Strike | large AoE burst and uninterruptible feel | slow/telegraphed compared with faster melee burst tools |
| Kingmaker | strong burst plus utility/knock-up value | exact mechanics and escalation behavior should be patch-verified |

## Burst melee tank fit

Burst melee tends to prefer clump tools that are already inside the melee front:

- Hand of Justice-like clump tools,
- 1h Mace-like clump tools,
- Heavy Mace hard-stop/support value,
- less reliance on ranged clump tanks like Camlann.

## Pressure / brawler tools from the episode

| Tool | Teaching value | Caution |
|---|---|---|
| Halberd | spreads bleed pressure and strains healers over time | not glamorous; value shows over duration |
| Infernal Scythe | can be clap/execute, but Q/W usage can add pressure value | avoid treating it only as an E weapon |
| Realmbreaker | versatile: pierce, pressure, bruiser, or burst-adjacent depending setup | exact build role must be stated clearly |
| Heron Spear | utility DPS: damage plus stun/control value | not chosen only by raw damage comparison |
| Clarent Blade | pressure weapon candidate after changes | patch-sensitive; do not overclaim without verification |

## Soft pressure concept

Soft pressure means useful damage and threat outside the main E engage.

The episode's core warning:

> If a zerg only has burst windows and no soft pressure, it may fail against groups that survive the engage and then punish the reset.

## Healing implication

Burst-dominated fights often increase the value of Fallen-style burst healing, but pressure/brawler fights increase the importance of Nature and sustained healing tools.

This supports the public healer doctrine that Holy and Nature both matter.
