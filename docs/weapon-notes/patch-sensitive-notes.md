# Current-Value Checks

The manual should not be afraid to use weapon and item examples. Examples help players understand how tools work.

Current-value checks matter when the guide makes claims about exact values, current behavior, or current strength.

## Check current information for

- cooldown values
- exact ability numbers
- item ability behavior after patches
- current strength or meta claims
- item tier/enchantment recommendations
- build substitutions
- weapon-specific interactions

## Usually stable lessons

- do not clump
- damage must land on time
- healer range matters
- defensive layers should cover the danger area
- support utility should align with damage
- tanks should stop enemy paths before damage lands
- terrain changes fight value
- movement is the path to a useful position
- tools should create value for the zerg, not only personal safety or meter value

## Better wording

Use:

> This tool can illustrate area denial and path control.

Avoid:

> This is permanently the best tool and every zerg should run it.

Use:

> Check current cooldown and item behavior before using this as a build recommendation.
