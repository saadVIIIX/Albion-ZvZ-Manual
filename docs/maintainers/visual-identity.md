# Visual Identity Direction

This page defines the early visual direction for the public website.

## Goal

The site should feel like a serious MMO doctrine/wiki hub, not a flashy recruitment page and not a plain text dump.

## Visual tone

- clean
- readable
- tactical
- calm
- documentation-first
- Albion-themed without copying Albion assets unnecessarily

## Current palette

| Use | Color |
|---|---|
| Primary blue | `#1f4e78` |
| Deep teal | `#0f5b78` |
| Warning orange | `#c75b12` |
| Light panel | `#eef6fb` |
| Background | `#f8fafc` |
| Border | `#b7c9d6` |

## What comes later

- polished diagrams
- item icons
- build-card visuals
- better homepage cards
- public branding/banner
- possibly guild-neutral logo treatment

## What to avoid

- over-designed pages that reduce readability
- using random Albion screenshots with private names
- making the site look like a build-list aggregator
- adding visuals before the teaching point is clear
