# v2.0 Milestone Notes

Website v2.0 is a beta-readiness milestone.

## What v2.0 represents

- Public-facing structure is stable enough for trusted review.
- Role pages and fight concepts are no longer just placeholders.
- Practical examples include review questions and drills.
- Source/credit/privacy/release readiness pages exist.
- Workbook direction is documented as a community template model.
- The project explicitly avoids becoming a guild-specific Brave/EVE clone.

## What v2.0 does not represent

- final visual polish
- final item icons
- final public workbook copy
- complete mechanics verification
- complete comp ecosystem

## Next phase after v2.0

The next phase should be about review and controlled polishing:

1. trusted review
2. feedback triage
3. public workbook copy creation
4. mechanics verification
5. visual/build-card asset pass
6. hosting/publishing workflow
