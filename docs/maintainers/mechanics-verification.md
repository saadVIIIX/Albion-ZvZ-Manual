# Mechanics Verification Plan

Albion values and item interactions can change. The site should avoid becoming outdated by separating field behavior from exact values.

## Verification sources

- Albion Wiki mechanics pages
- official patch notes
- in-game tooltip checks
- trusted current player review

## Page labels

Each page should use patch sensitivity:

- **Low** — broad doctrine or behavior unlikely to change quickly
- **Medium** — playstyle/build-adjacent information
- **High** — exact mechanics, values, item behavior, build-specific instructions

## What to verify before publication

- AoE escalation values and caps
- focus fire / resilience protection behavior
- resilience penetration details
- disarray/homesickness-type mechanics
- healing sickness behavior
- overcharge/IP behavior
- item abilities and cooldowns
- build-specific ability choices

## Safe wording pattern

Bad:

> AoE escalation always does exactly X.

Better:

> AoE escalation makes clumping more dangerous. Check current values before publishing hard numbers.
