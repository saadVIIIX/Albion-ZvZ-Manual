# Publication Go / No-Go Decision

Use this page before publishing a wider beta or stable release.

## Go

Publish or share wider if:

- homepage explains the project clearly
- Sources and Credit page is accurate
- known limitations are visible
- private content is removed/anonymized
- workbook link points only to a public-safe copy or is marked pending
- build cards are labeled draft/workbook-backed/patch-sensitive
- mechanics claims are verified or softened
- reviewer feedback has been triaged

## No-go

Do not publish wider if:

- master workbook might be exposed
- private VOD/player/guild details remain
- major role pages contain known errors
- mechanics pages include unverified hard numbers
- credit is unclear
- build cards look like universal meta prescriptions
- future packages look active when they are placeholders
- navigation cannot guide a new player

## Decision record

When a release decision is made, record:

```text
Version:
Date:
Decision: Go / No-go
Reason:
Required fixes before next attempt:
Reviewer input used:
```
