# Internal Linking Standard

Internal links turn the website from a pile of pages into a learning path.

## Link types

- **Prerequisite links**: what to read first
- **Related concept links**: concepts that support the page
- **Workbook links**: practical tool pages
- **Verification links**: wiki/patch-sensitive references
- **Training links**: drills that teach the concept

## Good page ending

Each major page should eventually end with:

```text
Related pages:
- Role guide
- Fight concept
- Practical example
- Training drill
- Workbook package, if relevant
```

## Avoid

- overlinking every keyword
- linking to draft pages as if they are stable
- hiding workbook-specific instructions inside general doctrine pages
