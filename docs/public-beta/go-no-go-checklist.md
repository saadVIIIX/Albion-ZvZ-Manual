# Public Beta v3.0.4 Go / No-Go Checklist

Use this before sharing the public beta link or package.

## Go

Public Beta v3.0.4 is shareable if:

- the homepage clearly says Public Beta, not Beta Candidate
- the guide's teaching purpose is clear before credits/source policy
- the guide is framed as public Albion ZvZ education
- Khelron / AO University credit is clear without overwhelming the reader path
- private VOD critique is not exposed with named guild members
- the workbook is clearly optional
- Great Fire Control Formation is clearly one applied package
- future comp packages are not promised without source data
- weapon examples teach function rather than pretending to be mandatory current-meta recommendations
- patch-sensitive mechanics are not presented as current verified facts unless verified
- known limitations are visible

## No-go

Do not share broadly if:

- the homepage or navigation still implies this is only an internal BC1 review package
- the source/credit framing overwhelms the manual's teaching purpose
- workbook/control-comp material appears to define the entire public manual
- private VOD critique appears as public named criticism
- item values or current meta claims are presented without verification
- the workbook copy being shared contains private names or officer/admin material

## Release decision

Public Beta v3.0.4 can be shared as a beta learning resource, not as a finished final manual.
