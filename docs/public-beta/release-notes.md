# Public Beta v3.0.4 Release Notes


## v3.0.3 Role Library Placement Fix

- Restored the **Role and Tool Library** to the main **Weapon and Ability Notes** navigation because it is useful as an advanced reader reference.
- Kept source-extraction batch material out of the main reader path; the problem was the recovered-note/source-recovery voice, not the functional role/tool library itself.
- No new doctrine, transcripts, comp packages, or meta claims were added.

## Release identity

**Albion ZvZ Manual - Website v3.0.4 Public Beta Naming Pass**

This release keeps the v3.0 Public Beta structure and applies targeted review feedback. It is a clarity pass, not a doctrine expansion pass.

## What changed from v3.0 Public Beta

- Homepage now leads with the manual's purpose: helping the general Albion playerbase learn ZvZ.
- Credits and source language now support the teaching mission instead of dominating the first impression.
- Sources / Credit page now explains that normal examples do not need individual source labels unless confidence, privacy, or patch sensitivity matters.
- Movement and Positioning page now clearly separates **positioning as the destination/state** and **movement as the path/adjustment**.
- Weapon and Ability Notes now emphasize tool function: how weapons/items create value, not what the current meta list is.
- Patch-sensitive language is narrowed toward exact values, cooldowns, current item behavior, and current strength claims.
- Review route updated to reflect the new clarity checks.

## What did not change

- No new transcript/source expansion was added.
- No new comp package was created.
- No new current-meta claims were introduced.
- The Great Fire Control Formation workbook remains the flagship full-doctrine applied package.
- Future comp packages remain community-submitted and source-dependent.
- Workbook V5.1 BC1 Review Fix remains the current workbook baseline until a separate public-copy pass.

## Current limitations

- Build-card visuals and item icons are still not final.
- The public workbook copy is still pending clean-copy review.
- Exact mechanics and current item values remain patch-sensitive when the guide makes specific claims.
- Some deep weapon/tool notes are source-backed but still require future verification and polish.
- This project is based primarily on Khelron / AO University teachings and does not claim to represent every ZvZ school of thought.

## Public beta rule

After this release, changes should be feedback-driven:

- fix blockers
- fix confusing public wording
- fix broken links/navigation
- verify patch-sensitive mechanics when needed
- avoid adding new source branches until there is a clear gap or blocker
