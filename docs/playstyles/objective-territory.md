# Objective / Territory Fighting


Objective and territory fights are shaped by geometry.

## What this style wants

It wants to control walls, gates, doors, repair paths, choke lines, and bomb angles.

## Key questions

- Which wall/gate gives the enemy best bomb angle?
- Where can repairers be hit?
- Where does the zerg retreat if counter-bombed?
- Which side forces the enemy into predictable movement?
- What area actually needs protection?

## Common mistake

Standing on a wall/side/objective area simply because it exists, not because it needs to be held.
