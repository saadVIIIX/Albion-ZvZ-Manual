# Public Sharing Note Template

Use this text when sharing a beta preview with trusted reviewers.

```text
This is a trusted-review beta preview of the Albion ZvZ Manual.

It is a public learning hub based on AO University/Khelron teachings, organized into role guides, fight concepts, practical examples, training pages, and workbook-supported comp packages.

Please pick the review lane closest to your experience:
- new player / line member
- healer
- DPS
- tank/support
- officer/caller
- mechanics/build review

Please focus on:
1. Is it understandable?
2. Is anything wrong or unsupported?
3. Is anything private/sensitive?
4. Can you find the page you need?
5. What should be fixed before public beta?

Known limitations:
- build cards are draft/workbook-backed
- item icons are not final
- diagrams are draft teaching visuals
- future comp packages are placeholders
- public workbook link may be pending until a cleaned copy exists
```

## Attachment rule

For trusted review, send:

- Website Preview ZIP
- Public Beta Review Pack
- optional Release Readiness packet

Do not send the master workbook unless the reviewer is part of the project team.
