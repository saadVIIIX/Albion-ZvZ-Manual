# Beta Milestone Hub

This section organizes the project into a stable beta milestone package.

The goal is not final release. The goal is to make the current website/workbook/archive set easy to review, store, and share with a small trusted group.

## What beta means here

Beta means:

- the structure is stable enough to review
- major learning paths exist
- role pages and fight concepts are usable
- practical examples have review questions and drills
- workbook public-copy strategy exists
- credit/privacy limitations are documented
- build cards are draft/workbook-backed, not final meta claims

Beta does **not** mean:

- every page is final
- item icons are complete
- the workbook is ready for broad public edit access
- mechanics values have all been verified for stable release
- future comp packages are active
