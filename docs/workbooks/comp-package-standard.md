# Comp Workbook Package Standard

Future comp workbooks should use a consistent package structure so players can move from the website into the correct live tool without confusion.

## A complete comp package should include

- playstyle identity
- intended scale
- party structure
- required healer package
- required support package
- build library
- coverage checklist
- role quick notes
- training drills
- common failures
- source/status notes
- live workbook link

## Why this matters

The website can eventually host several comp packages. Without a consistent standard, each workbook becomes its own confusing world.

A player should be able to open any comp package page and immediately answer:

1. What is this comp trying to do?
2. Who is this comp for?
3. What roles are required before luxury DPS?
4. Where is the live workbook?
5. What should I read before signing up?
