# Future Workbook Packages

Future workbooks should not be created just because a playstyle exists. They should be created when there is enough reliable doctrine, build data, and operational need.

A future workbook package should include:

- playstyle identity
- role package
- build library
- scaling path
- coverage checklist
- role quick notes
- training drills
- common mistakes
- website page link

The website should act as the hub that points players toward the correct workbook for the content being run.
