## What the ledger is for

The Formation Ledger is optional. The Albion ZvZ Manual should still be useful even if someone never opens the sheet.

Use the ledger when your group wants a practical place to:

- collect signups
- assign parties from one source of truth
- check healer, support, tank, and DPS coverage
- keep build references in one place
- run role notes, drills, and review checklists

## Current live ledger package

| Package | Where details live | Public meaning |
|---|---|---|
| Great Fire Control Formation | inside the Formation Ledger | active applied package and flagship example, not the whole manual |

Comp-specific structure, build library details, and party assignment logic belong inside the ledger itself. The public manual keeps the general teaching path separate so readers are not forced into one formation.

## How to use the ledger

1. Open the Google Sheets ledger.
2. Use **File -> Make a copy** before editing.
3. Add players on the **Signups** tab.
4. Assign party and slot from the Signups tab.
5. Let Party 1, Party 2, and dashboard views fill automatically.
6. Use coverage and role notes to fix missing healer/support/tank structure before content.
