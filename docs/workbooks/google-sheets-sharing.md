# Google Sheets Sharing Checklist

This checklist is for the moment the workbook becomes a shareable Google Sheets tool.

## Before uploading

- Confirm the workbook opens in Excel.
- Confirm no formula error strings are visible.
- Confirm dropdowns still work.
- Confirm no white text on light cells.
- Confirm tabs have sensible names.
- Confirm no private names remain.

## After uploading to Google Sheets

Check:

- frozen rows still make sense
- column widths are readable
- merged cells did not break layout
- dropdowns imported correctly
- formulas calculate
- protected/private tabs are not exposed
- sharing permissions are correct

## Sharing settings

For a public resource, likely use:

- Viewer access for the public copy
- Separate editable officer copy for live use
- Never share edit access to the master workbook publicly

## Website relationship

The website should link to the public workbook copy, not the private master workbook.
