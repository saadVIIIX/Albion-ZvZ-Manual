# Great Fire Control Support Package

The support package exists to keep the zerg alive and make damage matter.

## Critical support ideas

- Locus and similar cleanses/defensives answer real pressure.
- Oathkeepers and resistance support need to be near the actual danger area.
- Rootbound or similar defensive tools should be timed for enemy counter-pressure.
- Support DPS must land pierce, anti-heal, or utility before or with damage.

## Common failure

Supports follow the caller's literal movement instead of the enemy's real damage path. This creates safe-but-useless support positioning.

## Workbook behavior

The workbook should make critical supports visible before officers fill luxury DPS.
