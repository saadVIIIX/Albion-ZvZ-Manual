# Great Fire Control Healer Package

The current workbook treats healer coverage as a package, not only a count.

## Operating principle

Holy and Nature should both be valued.

Holy provides direct stabilization and burst recovery. Nature provides sustained coverage, HoT/zone value, and fight-stabilizing support that many zergs neglect.

## Why this is visible in the workbook

Many Albion zergs over-stack one direct-healing style and under-value Nature. This project intentionally keeps Nature visible so officers and players understand that sustained healing value matters.

## Workbook behavior

The workbook should track:

- healer count
- Holy coverage
- Nature coverage
- danger-area healing notes
- whether healers can reach the tank/front/choke area
- whether players actually stand in healing zones

## Public wording

The public site should say "Holy + Nature healer package" without pretending one fixed ratio is universal for all comps and patches. The workbook can enforce its own operating standard for this comp.
