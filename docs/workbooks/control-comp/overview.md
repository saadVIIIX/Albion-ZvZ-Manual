# Great Fire Control Formation

This is the first workbook package connected to the website.

## Identity

The current workbook supports a Great Fire / Brimstone-style pressure-control package. It should be treated as a practical operating tool for content, not as a universal public meta claim.

## What this comp wants

- stable healer package
- strong Holy + Nature coverage
- ranged pressure
- support-DPS timing
- heal cut
- pierce / utility before damage
- spacing as the first defensive layer
- defensive support on the actual danger area
- plate players creating space instead of hiding in friendly DPS

## What belongs in the workbook

- signups
- party assignment
- coverage checks
- build library
- Great Fire comp guide
- role quick notes
- training curriculum
- VOD review checklist

## What belongs on the website

- general doctrine
- role pages
- build-card summaries
- workbook package overview
- source/credit notes
- links to the live workbook when ready

## Status

This package is active as the current project workbook, but the public website should not present it as the only valid way to play ZvZ.
