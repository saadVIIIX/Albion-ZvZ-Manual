# Great Fire Control Party Structure

This page explains the party structure at a high level. The live workbook remains the source of truth for actual assignments.

## Party logic

The workbook uses party sheets to assign core roles first:

- caller / engage tools
- healer package
- defensive support
- support DPS
- ranged pressure
- melee pressure / execute
- flex tank slots

## Core before luxury

A zerg missing healers, defensive layers, pierce/heal-cut, or tank structure may look strong on paper but fail in the fight.

The workbook should help officers see these gaps before leaving.
