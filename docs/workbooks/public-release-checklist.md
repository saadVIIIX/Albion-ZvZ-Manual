# Workbook Public Release Checklist

Use this checklist before sharing the workbook beyond trusted friends.

## Privacy

- No private player names.
- No internal critique.
- No unlisted VOD notes.
- No guild-member criticism.
- No hidden sensitive notes in copied cells.

## Functionality

- Signup dropdowns work.
- Backup role dropdowns work.
- Coverage checklist reads correctly.
- Build Library is understandable.
- Website Hub points to the right version.
- No formula errors.

## Readability

- Home page is clean.
- Tabs are not chaotic.
- Text is readable in Google Sheets.
- Important pages have clear first-screen instructions.
- Support Priority / small tabs are compact enough.

## Doctrine

- Healer package emphasizes Holy + Nature.
- Role notes match current website pages.
- Great Fire comp-specific notes stay attached to the workbook package.
- Public manual concepts are not mixed with private source notes.

## Release state

Mark the public copy as:

- Draft
- Trusted-review copy
- Public beta
- Stable public copy
