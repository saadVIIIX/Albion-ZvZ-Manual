# Practical Example Review Prompts

Use these prompts when reviewing practical example pages.

## Universal example review questions

1. Is the situation common enough to matter?
2. Is the correct response clear?
3. Is the common failure specific?
4. Does the example teach a decision instead of only describing a mistake?
5. Is the drill practical?
6. Does it avoid naming private players?
7. Does it link to the right role/fight concept pages?
8. Would a role lead know how to use this in VOD review?

## Good feedback

Good feedback says:

```text
On the Kite as a Unit example, the role table is useful, but the support row should mention covering the last players through the danger path.
```

Bad feedback says:

```text
Kite page weird.
```

## Triage

Turn feedback into:

- fix now
- backlog
- needs source verification
- needs high-level review
- out of scope
