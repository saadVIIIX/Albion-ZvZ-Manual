# Known Limitations

This page should stay visible during trusted review and public beta.

## Current limitations

- Some pages are more polished than others.
- Build cards are still text-first.
- Item icons are not implemented yet.
- The current diagrams are draft teaching visuals, not final art.
- Mechanics values still need final verification before stable release.
- The Great Fire Control Formation package is workbook-backed, not a universal meta claim.
- Future comp packages are placeholders only unless marked active.
- The static Preview ZIP is a simplified review preview, not the final MkDocs Material experience.
- The public workbook copy has not been fully separated from the master workbook yet.

## Important design choices

These are not bugs:

- The site is for the general Albion playerbase, not one guild.
- The workbook is a copyable operating tool, not a forced membership system.
- Brave/EVE references are used for structure, not copied as guild infrastructure.
- AO University/Khelron remains the doctrine backbone.
- The project keeps depth even when some pages are long.

## Before public beta

Still needed:

- trusted role review
- mechanics/build sanity check
- public-copy workbook pass
- final source/credit check
- privacy pass
- public beta release note
