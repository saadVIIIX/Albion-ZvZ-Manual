# Known Limitations

The manual is live and usable, but it is not the final stable edition.

## Current limitations

- Some pages are more polished than others.
- Build cards are still mostly text-first.
- Item icons are not implemented yet.
- The current diagrams are teaching visuals, not final art.
- Exact mechanics values should still be verified when they matter for a current build or patch-sensitive claim.
- The Great Fire Control Formation is a workbook-backed applied package, not a universal meta claim.
- Future comp packages should remain placeholders unless they have enough source-backed data.
- Some deeper weapon/tool notes preserve source-backed examples that can be refined in a later source integration pass.

## Important design choices

These are intentional:

- The site is for the general Albion playerbase, not one guild.
- The Formation Ledger is a copyable operating tool, not a forced membership system.
- Brave/EVE-style references are used for structure and documentation habits, not copied as guild infrastructure.
- AO University/Khelron remains the doctrine backbone.
- The project keeps depth even when some pages are long.

## What belongs after the current public cleanup

- deeper integration of remaining transcript batches
- final build-card visuals and consistent item icon policy
- more diagrams or visual examples
- broader comp-package support when reliable community data exists
- possible move to a project GitHub organization for a cleaner free URL

## What should not block current use

The manual does not need every possible Albion weapon, comp, or meta branch before being useful. Its main value is teaching role function, decision-making, movement, positioning, pressure, defensives, terrain, and review habits.
