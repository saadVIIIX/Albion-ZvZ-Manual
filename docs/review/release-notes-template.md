# Release Notes Template

Use release notes when publishing or sharing a new milestone.

```text
Albion ZvZ Manual — Public Beta vX

What this is:
- Public ZvZ learning hub based on AO University/Khelron doctrine.
- Includes role guides, fight concepts, playstyle modules, training pages, and workbook links.
- The workbook is a copyable operating template, not a guild-specific requirement.

What changed:
- ...

Known limitations:
- Some build cards are draft/workbook-backed only.
- Mechanics values should be checked against current Albion Wiki/patch notes.
- Future comp packages are placeholders unless marked active.
- Public workbook link may be pending until a safe copy exists.

Credits:
- Core educational credit to Khelron and AO University.
- Additional organization, workbook structure, diagrams, and website workflow by the project maintainer team.

Feedback request:
- Please review role pages, mechanics wording, privacy, and navigation.
```

## Release-note tone

Release notes should sound useful and honest. Do not oversell the project as finished if it is still a beta.
