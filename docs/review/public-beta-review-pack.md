# Public Beta Review Pack

This is the practical review package for trusted friends before wider public release.

## What to send reviewers

Send:

- Website Preview ZIP
- a short explanation of what the project is
- reviewer instructions
- feedback form template
- known limitations
- review priority list

## Suggested message

```text
This is an early trusted-review preview of the Albion ZvZ Manual project.

It is based around AO University/Khelron teachings and is being built as a public ZvZ learning hub plus workbook companion.

Please do not review every page. Pick the lane closest to your role:
- new player / line member
- DPS
- healer
- tank/support
- officer/caller
- mechanics/build review

Main thing I need:
1. Is it understandable?
2. Is anything wrong?
3. Is anything private/sensitive?
4. Can you find the page you need?
```

## Review order

1. [Known Limitations](known-limitations.md)
2. [Reviewer Instructions](reviewer-instructions.md)
3. [Review Priority List](review-priority-list.md)
4. Relevant learning path
5. Relevant role/fight pages
6. [Feedback Form Template](feedback-form-template.md)

## What not to ask yet

Do not ask reviewers to judge the final visual polish yet. The site is improving visually, but the current priority is structure, doctrine accuracy, privacy, and navigation.

## First-page route

For the first review pass, use [First-Page Review Route](first-page-review-route.md). This prevents reviewers from opening random deep pages before they understand the project.
