# Trusted Review Hub

This section is for trusted reviewers before public release.

The goal is not to collect random opinions. The goal is to find:

- doctrine errors
- unclear explanations
- navigation problems
- privacy issues
- mechanics values that need verification
- workbook connection problems

## How to review

1. Pick one reader path.
2. Browse only the relevant pages first.
3. Mark confusing sections.
4. Separate doctrine disagreement from wording issues.
5. Do not try to review all 100+ pages in one sitting.

## Best reviewer types

- new player / normal line member
- experienced DPS
- healer main
- tank/support player
- caller/officer
- mechanics/build reviewer
- trusted AO University/Khelron-adjacent reviewer, if available

## What feedback should look like

Good:

> On the Nature Healer page, the argument is clear, but the examples should explain where the zone should be placed during a kite.

Less useful:

> Healer page feels weird.
