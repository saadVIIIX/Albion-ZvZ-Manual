# Feedback Form Template

Copy this template when asking for feedback.

```text
Reviewer name / role:
Date:
Pages reviewed:

1. What page was easiest to understand?

2. What page was hardest to understand?

3. Did anything seem wrong or unsupported?

4. Did anything feel too vague?

5. Did anything feel too long or badly organized?

6. Did the navigation make sense?

7. Did the Khelron / AO University credit feel clear and respectful?

8. Did any private/guild-sensitive detail seem exposed?

9. What one page should be improved first?

10. Any other notes?
```

## Triage rule

Feedback should become decisions, not scattered comments. Every useful comment should be turned into one of:

- fix now
- backlog
- needs source verification
- needs Khelron/high-level review
- rejected / out of scope
