# Reviewer Instructions

Use this page when sharing the preview with trusted friends.

## Before you start

Please remember:

- This is a draft, not final publication.
- The goal is public teaching, not guild drama.
- Private VOD examples are intentionally anonymized.
- The current Albion ZvZ Formation Ledger is one package, not the only valid Albion comp.

## Pick one review lane

### New player review

Can you find what to read first? Are the first rules understandable?

### Role review

Does your role page explain what to do before, during, and between engages?

### Officer review

Can you understand how to use the site with the workbook?

### Mechanics review

Are any mechanics claims too specific or likely outdated?

### Wording review

Does anything sound too harsh, too vague, or too insider-only?

## What to report

- page name
- section heading
- issue type
- suggested fix, if you have one
- whether it is critical or optional
