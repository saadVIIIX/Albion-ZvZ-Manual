# Fight Concept Review Prompts

Use these prompts when asking experienced players or officers to review fight concept pages.

## Universal concept review questions

1. Does the page explain the behavior clearly?
2. Can a line member understand the first half?
3. Can a role lead use the second half for review?
4. Are examples practical instead of abstract?
5. Are common failures specific enough?
6. Does it avoid overclaiming one comp as universal?
7. Are related pages useful?
8. Is any claim patch-sensitive or mechanics-specific?
9. Would this page produce a useful drill?
10. What one section should be improved first?

## Concept-specific prompts

### Movement and Positioning

- Does it explain role-specific movement?
- Does it warn against literal call-following?
- Does it explain spread without disconnecting?

### Counter-Engage Windows

- Does it explain who is safe enough to turn?
- Does it warn against panic-kiting?
- Does it connect to DPS timing and support stabilization?

### Defensive Layering

- Does it define danger area clearly?
- Does it explain support/healer overlap?
- Does it avoid treating defensives as random buttons?

### DPS Timing

- Does it make the sequence clear?
- Does it explain late damage clearly?
- Does it warn against waiting for perfect clumps?

### Terrain and Geometry

- Does it explain why terrain changes decisions?
- Does it connect tanks, supports, healers, and DPS to the route?
- Does it avoid making objective fighting too guild-specific?
