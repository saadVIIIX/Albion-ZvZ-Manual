# Feedback Triage Board

Use this page to turn feedback into decisions.

## Triage categories

| Category | Meaning | Action |
|---|---|---|
| Fix now | Clearly wrong, confusing, or risky | Patch next version |
| Backlog | Good idea, not urgent | Store for later |
| Needs source verification | Could be right, but needs evidence | Check transcripts/wiki/patch notes |
| Needs high-level review | Requires caller/Khelron-level input | Do not guess |
| Out of scope | Not aligned with project direction | Decline or archive |
| Already handled | Feedback matches existing plan | Note and close |

## Feedback record format

```text
Date:
Reviewer role:
Page:
Issue:
Category:
Decision:
Next action:
```

## Important rule

Feedback should not become scattered comments in Discord or chat. Every useful note should become one of the triage categories above.
