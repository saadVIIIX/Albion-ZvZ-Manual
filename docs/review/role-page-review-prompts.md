# Role Page Review Prompts

Use these prompts when asking role mains to review a role page.

## Universal role review questions

1. Does the page explain the role's purpose clearly?
2. Does it explain where the role should stand?
3. Does it explain what the role should watch?
4. Does it explain what happens before the engage?
5. Does it explain what happens during enemy pressure?
6. Does it explain what happens between engages?
7. Are common failures specific enough?
8. Are related pages useful?
9. Is anything wrong, outdated, or overclaimed?
10. Would a player in this role know one thing to improve next CTA?

## Healer-specific prompts

- Does the page explain the danger area clearly?
- Does it respect both Holy and Nature?
- Does it avoid treating healer value as only raw healing numbers?
- Does it explain where healing zones/direct heals should be useful?

## Tank/support prompts

- Does the page explain proactive positioning?
- Does it separate engage, stopper, and defensive responsibility clearly enough?
- Does it warn against healthy plate hiding in friendly DPS?
- Does it explain staying until the zerg is safe?

## DPS/support-DPS prompts

- Does the page explain timing?
- Does it separate pressure from main engage damage?
- Does it explain why utility must land before or with damage?
- Does it avoid teaching meter-chasing?
