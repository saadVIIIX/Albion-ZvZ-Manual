# Navigation Cleanup Notes

Use this page to avoid endless restructuring.

## Navigation goal

A reader should be able to find the right path without understanding the whole project.

## Primary paths

| Reader | Path |
|---|---|
| New player | Home → Start Here → Pick Your First Role |
| Role player | Home → Role Guides → relevant role |
| Officer/reviewer | Home → Training → VOD Review / Concept-to-Drill |
| Weapon/tool learner | Home → Weapon and Ability Notes |
| Workbook user | Home → Workbooks and Tools |
| Project maintainer | Home → Audit / Beta / Maintainers |

## Cleanup principles

- Do not create new sections unless there is no proper home.
- Prefer improving overview pages over adding more branches.
- Keep workbook links mostly inside Workbooks, Build Cards, and Beta/Audit pages.
- Keep specific weapons mostly inside Weapon Notes or Applied Build Cards.
- Keep general doctrine pages focused on behavior.

## v2.3 navigation decision

From v2.3 onward, navigation changes should support publication, not novelty.
